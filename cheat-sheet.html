HTML Tag Reference

HTML Tags

Tag Name Description
<header> Header A container used for navigational links or introductory containing <h1> to <h6> headings
<nav> Navigation Used to define a block of navigation links such as menus and table of contents. It can be used inside of <header> or on its own
<main> Main Used to encapsulate the dominant content within a webpage
<footer> Footer Encapsulates the footer, or the content at the bottom of the page
<section> Section Defines the elements in a document, such as chapters, headings, or any other area of the document with the same theme
<article> Article Holds the contents that make sense on its own
<aside> Aside Used to mark additional information that can enhance another element but isn't required to understand the main content'
<figure> Figure Used to encapsulate media such as image, illustration, diagrams etc.
<figcaption> Fig Caption Describes the media in the <figure> tag
<audio> Audio Embeds audio content into a document
<video> Video Adds video to the website
<embed> Embed Embeds any media content including videos, audio files, and gifs from an external source

Attributes

Attribute Name Description
controls Controls Adds controls like start and pause
autoplay Autoplay Plays the video automatically
loop Loop Replays the video when it ends
